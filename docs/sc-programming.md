# Strength & Conditioning Programming Model

## Block Structure
- Selectable block length: 4 / 6 / 8 weeks
- Default: 6 weeks
- Deload week configurable (default week 4)

## Progression Model (Hybrid)
- App suggests progression
- Clinician can override

### Progression Rules
- Bodyweight / Isometric: increase time or reps linearly
- Dumbbell / Kettlebell: increase reps first, then load
- Barbell lifts: increase load within rep ranges

## Actual Tracking
- Per exercise, per session, per week
- Stores actual reps, load, and completion flag

## Athlete Bias
- Templates are neutral by default
- Coaches may bias toward unilateral/isometric manually
